unity二进制存档和读档

2019-03-11
Unity

源自siki学院里的一个读存档例子

这个游戏是一个九宫格打怪物的游戏，其中，九个格子分别存以0~8的Target Positions，每个格子有当前激活的0~3的怪物类型Monster Types 。

1、新建一个save类来保存序列

2、在GameManager.cs里实例化Save对象，并存储当前游戏状态信息。

3、二进制存档

需要的命名空间

  1. 创建一个二进制格式化程序
    BinaryFormatter bf = new BinaryFormatter();
  2. 创建一个文件流
    FileStream fileStream = File.Create(Application.dataPath + "/StreamingFile" + "/byBin.txt");
  3. 用二进制格式化程序的序列化方法来序列化Save对象,参数：创建的文件流和需要序列化的对象
    bf.Serialize(fileStream, save);
  4. 关闭流
    fileStream.Close();

4、二进制读档

  1. 创建一个二进制格式化程序
    BinaryFormatter bf = new BinaryFormatter();
  2. 打开一个文件流
    FileStream fileStream = File.Open(Application.dataPath + "/StreamingFile" + "/byBin.txt", FileMode.Open);
  3. 调用格式化程序的反序列化方法，将文件流转换为一个Save对象
    Save save = (Save)bf.Deserialize(fileStream);
  4. 关闭文件流
    fileStream.Close();

5、通过读档信息重置我们的游戏状态（分数、激活状态的怪物）
