移除链表中的倒数第n个结点

Posted on 2019-01-27 | In leetcode

leetcode第19题

问题

Given a linked list, remove the n-th node from the end of list and return its head.

Example:

Note:

Given n will always be valid.

Follow up:

Could you do this in one pass?

解

要求删除倒数第n个链表节点，难点在于获取“倒数第n个”，可以用两个指针，slow和fast相隔n个节点，当fast走到链表的末尾，则距离fast有n个距离的slow正好指向要删除的那“倒数第n个”。
