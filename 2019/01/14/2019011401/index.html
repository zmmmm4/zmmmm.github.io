查找两个数组的中间值

leetcode第4题

【leetcode第4题】
There are two sorted arrays nums1 and nums2 of size m and n respectively.
Find the median of the two sorted arrays. The overall run time complexity should be O(log (m+n)).
You may assume nums1 and nums2 cannot be both empty.

Example 1:
nums1 = [1, 3]
nums2 = [2]
The median is 2.0 Example 2:
nums1 = [1, 2]
nums2 = [3, 4]
The median is (2 + 3)/2 = 2.5
【思路】
首先理解什么是中间值，在统计学里面就是用于把一个数集分为两个等长的子集，一个子集总是比另一个大。当我们理解了中间值的使用，我们就和答案非常接近了。

首先把A从一个随机的位置i分成两个部分：
left_A | right_A
A[0], A[1], …, A[i-1] | A[i], A[i+1], …, A[m-1]
A有m个元素，因此有m-1种切割的方式， len(left_A) = i, len(right_A) = m - i
（i=0时，left_A为空；i=m，right_A为空）
同样的，把B从一个随机的位置j分成两个部分：
left_B | right_B
B[0], B[1], …, B[j-1] | B[j], B[j+1], …, B[n-1]
把A和B放在一起，则有：
left_part | right_part
A[0], A[1], …, A[i-1] | A[i], A[i+1], …, A[m-1]
B[0], B[1], …, B[j-1] | B[j], B[j+1], …, B[n-1]
按照中间值的含义，如果：
1) len(left_part) == len(right_part)
2) max(left_part) <= min(right_part)
那么median = (max(left_part) + min(right_part))/2.
具体表示就是：
(1) i + j == m - i + n - j
(or:如果m+n为奇数，i+j比m-i+n-j多1则
i + j == m - i + n - j + 1)
if n >= m, we just need to set: i = 0 ~ m, j = (m + n + 1)/2 - i
(为什么n>=m，因为i有可能取到m，若同时n<m则j将为负数，会得到无效解)
(2) B[j-1] <= A[i] and A[i-1] <= B[j]
因此我们应该做的是：
Searching i in [0, m], to find an object i that:
B[j-1] <= A[i] and A[i-1] <= B[j], ( where j = (m + n + 1)/2 - i )
这个搜索i的过程具体描述为：
<1> 设置imin = 0, imax = m, 在[imin, imax]里面展开搜索

<2> i = (imin + imax)/2, j = (m + n + 1)/2 - i

<3> 现在我们有len(left_part)==len(right_part).有三种情况：

   <a> B[j-1] <= A[i] and A[i-1] <= B[j]
       找到了i，可以停止搜索了。
   <b> B[j-1] > A[i]
       B的左侧最大值比A的右侧最小值大了，说明A[i]太小，需要向右移动i，i增大同时j会减小。
       要增大i的话，要调整搜索范围变成[i+1, imax]，因此设置imin = i+1，回到<2>.
   <c> A[i-1] > B[j]
       A的左侧最大值比B的右侧最小值大了，说明A[i-1]太大，需要向左移动i，i减小同时j会增大。
       要减小i的话，要调整搜索范围变成[imin, i-1]，因此设置imin = i+1，回到<2>.

i搜索到之后，中间值为：
m+n为奇数时：max(A[i-1], B[j-1])
m+n为偶数时：(max(A[i-1], B[j-1]) + min(A[i], B[j]))/2
接下来我们来考虑极限情况， i=0,i=m,j=0,j=n时，A[i-1],B[j-1],A[i],B[j]中的某些可能会不存在，因此我们就不用同时检查B[j-1] <= A[i]和 A[i-1] <= B[j].只检查其中一个就够了。
所以我们应该做的是：
Searching i in [0, m], to find an object i that:
(j == 0 or i == m or B[j-1] <= A[i]) and
(i == 0 or j == n or A[i-1] <= B[j])
where j = (m + n + 1)/2 - i
在一个搜索循环里，我们只会遇到三种情况：
(j == 0 or i == m or B[j-1] <= A[i]) and
(i == 0 or j = n or A[i-1] <= B[j])
找到了i可以停止了

<b> i < m and B[j - 1] > A[i]
     i太小，需要增大

<c> i > 0 and A[i - 1] > B[j]
     i太大，需要减小

（由于i < m ==> j > 0 和 i > 0 ==> j < n，因此和不用检查j）
【解】
def median(A, B):
m, n = len(A), len(B)
if m > n:
A, B, m, n = B, A, n, m
if n == 0:
raise ValueError
imin, imax, half_len = 0, m, (m + n + 1) / 2
while imin <= imax:
i = (imin + imax) / 2
j = half_len - i
if i < m and B[j-1] > A[i]:

    # i is too small, must increase it
    imin = i + 1
elif i > 0 and A[i-1] > B[j]:
    # i is too big, must decrease it
    imax = i - 1
else:
    # i is perfect
    if i == 0: max_of_left = B[j-1]
    elif j == 0: max_of_left = A[i-1]
    else: max_of_left = max(A[i-1], B[j-1])
    if (m + n) % 2 == 1:
        return max_of_left
    if i == m: min_of_right = B[j]
    elif j == n: min_of_right = A[i]
    else: min_of_right = min(A[i], B[j])
    return (max_of_left + min_of_right) / 2.0

class Solution {
public:
double findMedianSortedArrays(vector& nums1, vector& nums2) {
vector A=nums1,B=nums2,p;
int m=nums1.size(), n=nums2.size(),pass=0;
if(m>n){
pass=m;
m=n;
n=pass;
p=A;
A=B;
B=p;
}
int imin=0,imax=m;
int max_of_left=0,min_of_right=0;
int i=0,j=0;
while (imin <= imax ){
i = (imin + imax) / 2;
j = (m + n + 1) / 2 - i;
if(i < m && B[j-1] > A[i]){
imin=i+1;
}
else if( i > 0 && A[i-1] > B[j]){
imax=i-1;

        }
        else{
            if(i==0){
                max_of_left = B[j-1];
            }
            else if(j == 0){
                max_of_left = A[i-1];
            }else{
                max_of_left = max(A[i-1], B[j-1]);
            }
            if ((m + n) % 2 == 1){
                return max_of_left;
            } 
            if(i==m){
                min_of_right = B[j];
            }
            else if(j == n){
                min_of_right = A[i];
            }else{
                min_of_right = min(A[i], B[j]);
            }
            return (max_of_left + min_of_right) / 2.0;
        }
    }
    return 0;

}
