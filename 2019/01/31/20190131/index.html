问题

Given n pairs of parentheses, write a function to generate all combinations of well-formed parentheses.

For example, given n = 3, a solution set is:

给出所有正确的括号组合。

解

先不考虑代码，用笔算的话要怎么解决这个问题？

N=3的话就会进行如下思考：

1、（

2、（）or （（

3、【（）（ 】or 【（（） or （（（】

4、【（）（（or （）（）】or【（（））or（（）（】or【（（（）】

可以总结一下规律：

  • 第一步先写（，下一步可能是（ 或者 ）
  • 当前左括号数>右括号数时，下一步才可以写 ）
  • 当左括号数==n时，不可再写（；右括号数==n时，结束。

这种每一步都是相同的操作，即+（ 或者 +）操作的题目，可以考虑使用递归的方法来解决。
